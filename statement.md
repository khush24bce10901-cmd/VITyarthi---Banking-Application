## Problem statement
Lightweight version of Banking Software that offers Security (password and amount) along with easy banking operations like withdraw/deposit

## Scope of the project
  * User Authentication (Registration/Login/Logout)
  * Transactions (Deposit/Withdraw)
  * Transaction History
  * Validation and Error Handling
  * 
## Target users
  * **Students and Learners:** People who want to learn basic functions of OOPs
  * **Instrcutors:** For showing secure transactions and validation processes as well as how to handle errors and maintain security
    
## High-level features
  * **Secure User Authentication:** Only allows registered users to access accounts details
  * **Real-Time Transactions**
  * **History Saving:** Automatically stores transactions for each user
