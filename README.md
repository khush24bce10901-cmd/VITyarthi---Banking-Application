# VITyarthi---Banking-Application

## Project Overview
This is a simple Banking System built in Java. It allows users to create an account, manage their money, and view transaction history.

## Features
* **User Management:** You can register a new user and login securely
* **Banking Operations:** Deposit and withdraw money with validation
* **History:** View a list of all your past transactions
* **Data Structure:** Uses a HashMap to store user data

## Technologies Used
* Java
* IntelliJ IDEA 

## How to Run
1.  Download the code or clone this repository
2.  Open the folder in your IDE
3.  Run the `Main.java` file
4.  Follow the instructions on the console

## How to Test
1.  Run the app
2.  Select **Option 1** to Register yourself
3.  Select **Option 2** to Login 
4.  Try **Option 3** to Deposit money
5.  Try **Option 4** to Withdraw withdraw.
6.  Select **Option 5** to see your Transaction History
7.  Select **Option 6** to Logout
