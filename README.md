# VITyarthi---Banking-Application

## Project Overview
This is a simple Banking System built in Java. It allows users to perform CRUD operations on an account, manage their money, and view transaction history.

## Features
* **User Management:** You can register a new user and login securely
* **CRUD Operations:** You can create account, get you transactional history, update your credentials(username and password), and delete your account if you wish
* **Banking Operations:** Deposit and withdraw money with validation
* **History:** View a list of all your past transactions
* **Data Structure:** Uses a HashMap to store user data

## Technologies Used
* Java
* IntelliJ IDEA 

## How to Run
1.  Download the code or clone this repository
2.  Open the folder in your IDE
3.  Run the `Main.java` file
4.  Follow the instructions on the console
